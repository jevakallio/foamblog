---

But this CSS makes it display the title instead of the content

```css
a[title] {
  font-size: 0;
}

a[title]:after {
  content: attr(title);
  font-size: initial;

  outline-width: 4px;
  outline-color: rgba(255, 255, 255, 0.8);
  outline-style: solid;
}
```
